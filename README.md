# Crypto-Monitor-App

#Sobre o aplicativo

Um aplicativo Android simples para monitorar cotações de criptomoedas usando a API do Mercado Bitcoin que exibe:
-Preço atual
-Valores de compra e venda
-Maiores altas e baixas do dia
-Volume negociado
